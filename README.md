# grade.calculator.c

Function Descriptions
inputScore() – Prompts the user to enter a quiz or exam score and returns the entered value.
calculateFinalGrade() – Computes the weighted final grade using 40% of the quiz score and 60% of the exam score.
getLetterGrade() – Determines the corresponding letter grade (A, B, C, D, or F) based on the final grade.
displayReport() – Displays the quiz score, exam score, final grade, and letter grade in a formatted report.
Program Flow

The program asks the user to enter the quiz and exam scores, calculates the weighted final grade, determines the corresponding letter grade, and displays all the results in a formatted grade report.
